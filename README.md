# GFY-CPP

An interpreter for the GFY language written in C++.

- [Building](#building)
- [Usage](#usage)
- [Aknowledgements](#aknowledgements)

## Building

To build the interpreter, run the following commands:

```bash
$ git clone https://github.com/LanaMirko04/gfy-cpp.git
$ cd gfy-cpp
$ make
```
